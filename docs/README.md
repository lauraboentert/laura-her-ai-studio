# Laura Böntert | Her AI Studio
## Claude Code Website Pack

Dieses Paket enthält:
- `STYLEGUIDE.md` – visuelles System, Farben, Typografie, Layout und Motion
- `CONTENT.md` – empfohlene Website-Struktur und fertige englische Copy
- `CLAUDE.md` – permanente Projektregeln für Claude Code
- `PROMPTS.md` – Master-Prompt und Schritt-für-Schritt-Prompts
- `SETUP.md` – technische Einrichtung, Assets, Tests und Deployment
- `site-content.ts` – strukturierte Inhalte als Ausgangspunkt für die Website

## Empfohlener Ablauf

1. Lege einen neuen Next.js-Ordner an.
2. Kopiere `CLAUDE.md` direkt in das Projekt-Root.
3. Lege Portrait und CV im `public`-Ordner ab.
4. Starte Claude Code im Projektordner.
5. Nutze zuerst den Master-Prompt aus `PROMPTS.md`.
6. Arbeite danach die Phasen-Prompts einzeln ab.
7. Lass vor dem Deployment den QA-Prompt laufen.
